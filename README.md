# AI-Powered Cyber Threat Intelligence Platform

A full-stack cybersecurity system that detects phishing URLs using AI, threat intelligence, and domain analysis.

## Features
- AI Phishing Detection (RandomForest)
- Threat Intelligence (OpenPhish, PhishTank)
- Domain Intelligence (WHOIS)
- SOC Dashboard
- Analytics
- Chatbot

## Tech Stack
- Flask (Backend)
- HTML/CSS/JS (Frontend)
- SQLite (Database)
- Scikit-learn (ML)

## Run

pip install -r requirements.txt
python train_model.py
python app.py
